The RSeQC module parses results generated by
[RSeQC](http://rseqc.sourceforge.net/),
a package that provides a number of useful modules that can
comprehensively evaluate high throughput RNA-seq data.

Supported scripts:

* `bam_stat`
* `gene_body_coverage`
* `infer_experiment`
* `inner_distance`
* `junction_annotation`
* `junction_saturation`
* `read_distribution`
* `read_duplication`
* `read_gc`
